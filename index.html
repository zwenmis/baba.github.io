Discord Bot Geliştirme Rehberi (Node.js)

Nasıl Başlanır?

İlk Discord botunu oluşturma adımlarını öğren. Discord Developer Portal'dan bot tokeni alma ve Node.js için gerekli **discord.js** kütüphanesini kurma.

Popüler Komutlar

Sık kullanılan komutlar için hazır kod örnekleri. Ping, avatar, rastgele sayı üretme gibi komutların kodlarını burada bulabilirsin.

Config Dosyası

Botunun ayarlarını (token, prefix) düzenli ve güvenli bir şekilde yönetmek için **.env** dosyasını nasıl oluşturacağını ve kullanacağını öğren.

Basit Node.js Bot Örneği

Bu örnek, `.env` dosyasından tokeninizi okuyan ve basit bir `ping` komutunu çalıştıran bir botun temel yapısını gösterir. 


// index.js
const { Client, GatewayIntentBits } = require('discord.js');
const dotenv = require('dotenv');

dotenv.config();

const client = new Client({ intents: [GatewayIntentBits.Guilds, GatewayIntentBits.MessageContent, GatewayIntentBits.GuildMessages] });
const prefix = '!';

client.once('ready', () => {
    console.log(`Bot ${client.user.tag} olarak giriş yaptı!`);
});

client.on('messageCreate', message => {
    if (!message.content.startsWith(prefix) || message.author.bot) return;

    const args = message.content.slice(prefix.length).trim().split(/ +/);
    const command = args.shift().toLowerCase();

    if (command === 'ping') {
        message.reply(`Pong! ${client.ws.ping}ms`);
    }
});

client.login(process.env.TOKEN);

.env Dosyası Örneği

Bot tokeninizi doğrudan kod içine yazmak yerine, bu dosyada güvenli bir şekilde saklamalısınız. 


// .env
